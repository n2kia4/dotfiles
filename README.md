# dotfiles

My configuration files.

## Setup

Open a terminal and run this:

```bash
bash -c "$(curl -LSs https://raw.github.com/n2kia4/dotfiles/master/setup.sh)"
```

The setup list is this:

- Download dotfiles
- Create symbolic links
- Install Homebrew
- Install packages
- Install Vim plugins
- Update Vim plugins
- System Preferences
- Change the login shell
